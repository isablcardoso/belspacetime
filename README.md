## 🖥️ Projeto

Esse é um projeto web responsivo de uma capsula para exibir memórias em uma linha do tempo.

## 🚀 Tecnologias

Esse projeto foi desenvolvido durante o NLWS da Rocketseat com as seguintes tecnologias:

- HTML
- CSS
- Git e Github
